# acquisitions
Predicting acquisitions based on Twitter data


## Dependencies
Install [dotenv](https://github.com/theskumar/python-dotenv)
```
$ pip install dotenv
```


Create a `.env` file and set your API keys and access tokens
```
CONSUMER_KEY    = '####'
CONSUMER_SECRET = '####'
TOKEN_ACCESS    = '####'
TOKEN_SECRET    = '####'
```
